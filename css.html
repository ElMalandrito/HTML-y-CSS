Notas CSS

    Selectores

  • Selector Universal (*):

    El selector universal, representado por "*", selecciona todos los elementos en la página web.

    Ejemplo: * { color: red; }

  • Selector de Tipo (o Selector de Elemento):

    El selector de tipo selecciona elementos por su nombre de etiqueta HTML.

    Ejemplo: p { font-size: 16px; }

  • Selector de Clase (.nombreDeClase):

    El selector de clase selecciona elementos que tienen una clase específica en su atributo "class".

    Ejemplo: .destacado { color: blue; }

  • Selector de Atributo ([atributo]):

    El selector de atributo selecciona elementos que tienen un atributo HTML específico.

    Ejemplo: a[target] { text-decoration: underline; } subrayará los enlaces que tienen el atributo "target".

  • Selector de Atributo con Valor ([atributo="valor"]):

    Este selector selecciona elementos que tienen un atributo específico con un valor específico.

    Ejemplo: input[type="text"] { border: 1px solid #ccc; } aplicará un borde a los elementos de entrada de tipo "text".

  • Selector de Atributo que Comienza con ([atributo^="valor"]):

    Este selector selecciona elementos cuyo atributo comienza con un valor específico.

    Ejemplo: a[href^="https://"] { color: green; } aplicará un color verde a los enlaces que comienzan con "https://".

  • Selector de Descendencia (espacio):

    El selector de descendencia selecciona elementos que son descendientes directos de otro elemento.

    Ejemplo: ul li { list-style-type: square; } aplicará viñetas cuadradas a todos los elementos <li> que son descendientes directos de una lista desordenada <ul>.
